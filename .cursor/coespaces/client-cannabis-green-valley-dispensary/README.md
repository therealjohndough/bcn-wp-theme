## Development Setup

### Prerequisites
- Docker and Docker Compose
- Node.js 16+
- PHP 8.1+

### Local Development
1. Copy environment template:
   ```bash
   cp .env.example .env
   ```

2. Start development environment:
   ```bash
   docker compose up --build
   ```

3. Access the site at http://localhost:8080
